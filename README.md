# Yatzy Game (Vue 3)
Welcome to the digital version of the Yatzy game, thas I developer as a free-time project using Vue 3 and the Composition API. This game offers a responsive and enjoyable experience, playable on both mobile and desktop devices. You can enjoy it in single-player mode or challenge your friends in multiplayer mode!

## Features
Single-player

Multiplayer: Copete against friends, who gets the best score?

Responsive Design: The game adapts seamlessly to both mobile and desktop screens for an optimal user experience.

Smooth User Experience: Developed with a focus on smooth and intuitive gameplay, making it fun and engaging for all users.

## Technologies Used
Vue 3: Front-end framework to build the application.

Composition API: For a more modular and readable code structure.

CSS: For styling and ensuring a responsive layout.

Jest: For test coverage.
